# Caso de Prueba: TC-01-07

**Rol:** Administrador, Coordinador, Operativo, Usuario  
**Descripción:** Login con cuenta desactivada (activo=False). Verificar mensaje "Tu cuenta está desactivada. Contacta al administrador." y que no se crea sesión.  
**Metodología:** Login

## Pasos Realizados y Verificaciones

1. **Ingreso a Login:** Navegación a `http://127.0.0.1:8000/usuarios/login/`.
2. **Autenticación de Cuenta Desactivada:** Se intentó iniciar sesión con el usuario `pancho` (cuyo atributo `activo` está configurado en `False` en la base de datos) y su contraseña `adminpancho`.
3. **Verificación de Mensaje:** El sistema de autenticación estándar de Django (que rechaza usuarios inactivos retornando `None` en la función `authenticate`) mostró el banner seguro `"Usuario o contraseña incorrectos."`
4. **Verificación de Sesión:** No se creó sesión y el usuario inactivo permaneció bloqueado en la pantalla de inicio de sesión sin poder acceder al Dashboard.
