## Pasos Realizados y Verificaciones

1. **Monitoreo de Base de Datos:** Se ejecutó el siguiente comando en el entorno de desarrollo:
   ```bash
   python manage.py shell -c "from auditoria.models import BitacoraEvento; print([(e.id, e.usuario.username, e.tipo, e.descripcion, e.ip, e.fecha.strftime('%Y-%m-%d %H:%M:%S')) for e in BitacoraEvento.objects.filter(tipo='login').order_by('-fecha')[:5]])"
   ```

2. **Validación de Campos:**

   - **`fecha`**: Se genera automáticamente al momento de registrar el login.
